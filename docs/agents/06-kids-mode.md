## 연령별 UI/UX 차이

| 요소 | 1세 | 3세 | 6세 |
|------|-----|-----|-----|
| **버튼 크기** | 100x100px | 80x80px | 60x60px |
| **색상** | 매우 밝고 원색 | 밝은 파스텔 | 파스텔 + 약간 성숙 |
| **텍스트** | 없음 (아이콘만) | 큰 글자 (20px+) | 보통 글자 (16px) |
| **네비게이션** | 없음 (자동 진행) | 하단 2~3 탭 | 하단 4 탭 |
| **콘텐츠** | 동요, 색상, 숫자 | 짧은 만화, 노래 | 교육 만화, 스토리 |
| **상호작용** | 탭으로 소리 재생 | 따라하기 게임 | 간단한 퀴즈 |
| **학습 방식** | 노출 (듣기 위주) | 반복 따라하기 | 단어 인식 + 퀴즈 |
| **시간 제한** | 15분 | 20분 | 30분 |
| **보상** | 소리 효과 | 스티커 | 스티커 + 별 + 레벨업 |

## 유아 모드 플로우

```
[부모 잠금 해제]
    │ PIN 4자리 또는 "이 숫자를 드래그하세요" (성인만 가능한 제스처)
    v
[프로필 선택]
    │ 큰 아바타 아이콘
    v
[연령별 홈 화면]
    ├── 1세: 큰 동물/색상 아이콘 → 탭 → 소리+영상 자동재생
    ├── 3세: 카테고리 아이콘 3개 → 영상 목록 → 자동재생 + 따라하기
    └── 6세: 카테고리 4개 + 오늘의 학습 → 영상 + 퀴즈
```

## 부모 잠금 (Parental Lock)

### 잠금 진입 시점
- 설정 화면 접근
- 유아 모드 ↔ 성인 모드 전환
- 시간 제한 해제 시도
- 앱 삭제 시도 (OS 레벨)

### 잠금 방식
- **PIN 4자리**: 기본
- **계산 문제**: "12 + 7 = ?" (유아가 풀 수 없는 수준)
- **텍스트 드래그**: "이 텍스트를 오른쪽으로 드래그하세요" (읽기 능력 필요)

## 화면 시간 관리

```typescript
// useKidsStore.ts
interface KidsState {
  dailyLimitMin: number;        // 일일 제한 (기본 20분)
  todayWatchedMin: number;      // 오늘 시청 시간
  lockAfterTime: string;        // "21:00" 이후 잠금
  isTimeLimitReached: boolean;  // 제한 도달 여부
  allowedCategories: string[];  // 허용 카테고리
}
```

### 시간 제한 도달 시
1. 현재 영상 종료 후 부드럽게 차단
2. "오늘은 여기까지! 내일 또 만나요 🌙" 화면
3. 부모 PIN으로만 해제 가능
